\# EV Charger Analytics



Automated EV charging station feasibility, ROI, and site scoring platform.



\## Features (v1)

\- Demand scoring

\- Competition analysis

\- ROI \& payback calculation

\- Go / No-Go verdict

\- Automated feasibility reports



\## Tech Stack

\- Python 3.12

\- OpenStreetMap

\- U.S. DOE AFDC API

\- U.S. Census Bureau (later)



\## Status

Initial MVP under development.
